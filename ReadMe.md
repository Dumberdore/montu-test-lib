# QuickRoute's Address Suggestion API 

## prerequisites
- Install NodeJs / NPM
```shell
# On MacOS, using Homebrew
brew install node@20
brew install npm
```
- Install Axios - https://www.npmjs.com/package/axios
```shell
npm install --save-dev typescript tsup vitest
```
